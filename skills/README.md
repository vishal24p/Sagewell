# Local Skills

This directory is the local skill source for Sagewell agents.

| Namespace | Purpose |
|---|---|
| `project/` | Project-specific skills and learnings. |
| `external/` | Vendored external skills copied into this repo. |

Read `../SKILLS.md` for routing rules. Prefer these local paths over outside installed skills. Do not edit files under `external/` unless explicitly syncing, upgrading, or repairing the vendored copy.
