# Glossary

## ACL

Access control list. A resource-level list of principals and permissions.

## Actor

The user or service account performing an action.

## Chunk

A searchable segment of a document with text, metadata, and optionally an embedding.

## Citation

A reference from an answer to the chunk or document used as evidence.

## Clean Architecture

A boundary pattern where domain and application logic do not depend on framework or infrastructure details.

## Hybrid Retrieval

Retrieval that combines dense vector search with lexical search.

## LangGraph

A workflow orchestration library used to model the answer pipeline as explicit state transitions.

## LlamaIndex

A library used for document ingestion, chunking, indexing, and retrieval utilities.

## Prompt Injection

An attempt inside user input or retrieved documents to override system instructions or security policy.

## RAGAS

An evaluation framework for retrieval-augmented generation quality metrics.

## RBAC

Role-based access control. Permissions are assigned through roles and sometimes scoped to resources.

## Single Tenant

One organization or deployment boundary. It does not remove the need for user-level access control.
